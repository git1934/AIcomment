# AIコメントサンプル（Streamlit試作）

評価するデータ断面・コメント枠・コメントの目的に応じて、生活面／学習面のコメントを自動生成するStreamlitアプリです。

## この版の変更点

- 画面タイトルを「AIコメントサンプル」に変更
- サイドバーの見出しを「AIコメント定義」に変更
- 「コメント形式」を「コメント枠」に変更
- コメントの目的をラジオボタンで選択する形式に変更
- 評価するデータ断面から「任意期間」を削除
- 集計結果の表示を削除
- 画面上部の説明文を削除
- 画面右側の設定内容表示を削除

## この試作でできること

- 評価するデータ断面の選択
  - 過去1週間の傾向
  - 過去1か月の傾向
  - 前週との比較
  - 前月との比較
- コメント枠の選択
  - 1枠：汎用的なコメント
  - 2枠：生活の様子コメント＋学習の様子コメント
- コメントの目的の選択
  - 汎用的なコメント
  - 不登校を防ぐためのコメント
  - 異常値を重視したコメント

## 使用する指標

### 生活の様子コメント

- 病気欠席数
- 事故欠席数
- 遅刻数
- 早退数
- 忌引等数
- 出席停止数
- 保健室利用数
- 心の天気晴れ数
- 心の天気曇り数
- 心の天気雨数

### 学習の様子コメント

- 総学習時間
- 解答数
- 正解率

## 起動方法

```bash
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

## 補足

この試作では生成AI APIを呼び出していません。Pythonの集計結果とテンプレート文でコメントを生成しています。
