# AIコメントアプリ（試作）

児童1人分の60日分ダミーデータをもとに、生活面・学習面のAIコメント風テキストを生成するStreamlitアプリです。
生成AI APIは使わず、Pythonの集計ロジックとテンプレートでコメントを生成します。

## 主な仕様

- タイトル：AIコメントアプリ
- 対象：A君へのAIコメント
- 評価するデータ断面：過去1週間、過去1か月、前週との比較、前月との比較
- コメント枠：1枠（汎用）／2枠（生活の様子・学習の様子）
- コメントの方針：汎用的なコメント、不登校を防ぐためのコメント、異常値を重視したコメント
- 使用指標：生活系・学習系の指標を画面上で選択可能
- コメントの種類：定性的なコメント／定量的なコメントをラジオボタンで選択可能
- コメント：定性的コメントは傾向中心、定量的コメントは数値・前期間比較を含む形式で生成
- 1枠表示では、生活・学習のどちらか一方だけを選んだ場合も、未選択側の警告文は表示せず、選択指標だけでコメント生成

## 起動方法

```bash
python -m pip install -r requirements.txt
python -m streamlit run app.py
```
