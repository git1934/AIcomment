# AIコメントアプリ

児童1人分のサンプルデータから、評価するデータ断面・コメント枠・コメント目的・使う指標を選び、AIコメントのサンプル文を表示するStreamlit試作アプリです。

## 起動方法

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

Windowsで `streamlit` コマンドが認識されない場合は、以下を使ってください。

```bash
python -m streamlit run app.py
```

## ファイル構成

```text
app.py
sample_data.csv
requirements.txt
README.md
```

## 仕様メモ

- データアップロード機能はありません。
- ダウンロード機能はありません。
- 生成AI APIは呼び出していません。
- コメント文はテンプレートロジックで生成しています。
- コメントはタイプライター風に1文字ずつ表示されます。
- 対象期間のデータは、画面下の「対象期間のデータを表示する」ボタンから表示します。
- 使う指標は、生活系指標・学習系指標それぞれで選択できます。

## コメント枠

- 1枠：汎用的なコメント
- 2枠：生活の様子コメント、学習の様子コメント

## 評価するデータ断面

- 過去1週間の傾向
- 過去1か月の傾向
- 前週との比較
- 前月との比較
