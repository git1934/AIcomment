# 日本語コメント生成アプリ（Streamlit試作）

指定した評価断面・コメント形式に応じて、生活面／学習面の日本語コメントを自動生成するStreamlitアプリです。

## この版の変更点

- 日本語CSVの文字化け対策を追加
- データアップロード機能を削除
- 画面左側の「対象児童」表示を削除
- 「サンプルデータを使用中です。」の表示を削除
- コメントCSVダウンロード機能を削除

## この試作でできること

- 評価するデータ断面の選択
  - 過去1週間の傾向
  - 過去1か月の傾向
  - 前週との比較
  - 前月との比較
  - 任意期間
- コメント形式の選択
  - 1枠：汎用的なコメント
  - 2枠：生活の様子コメント＋学習の様子コメント
- コメントの目的の選択
  - 汎用的なコメント
  - 早期支援につなげるコメント
  - 異常値を重視したコメント
- 集計結果と対象期間データの確認

## 使用する指標

### 生活の様子コメント

- 病気欠席数
- 事故欠席数
- 遅刻数
- 早退数
- 忌引等数
- 出席停止数
- 保健室利用数
- 心の天気晴れ数
- 心の天気曇り数
- 心の天気雨数

### 学習の様子コメント

- 総学習時間
- 解答数
- 正解率

## 起動方法

```bash
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

## 補足

この試作では生成AI APIを呼び出していません。Pythonの集計結果とテンプレート文でコメントを生成しています。
